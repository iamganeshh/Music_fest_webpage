# Music_fest_webpage
🎵 MusicFest 2026

A responsive and modern music festival website built using HTML, CSS, Bootstrap, and JavaScript.
This project showcases a music event landing page with event sections, ticket booking form, contact form, and interactive UI features.

🚀 Features
Responsive Navbar with smooth scrolling
Attractive Hero Section
Upcoming Events Cards
Ticket Booking Form
Booking Confirmation Message using JavaScript
Contact Form UI
Bootstrap Icons Integration
Fully Responsive Design using Bootstrap 5
🛠️ Technologies Used
HTML5
CSS3
Bootstrap 5
JavaScript
Bootstrap Icons
📂 Sections Included
Home
Events
Tickets
Contact
Footer with Social Media Icons
💡 Project Purpose

This project was created to practice:

Bootstrap Components
JavaScript DOM Manipulation
Form Handling
